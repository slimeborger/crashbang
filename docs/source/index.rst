Welcome to Crashbang's documentation!
=====================================

**Crashbang** is a simple, python app desgined to improve productivity by a more *heavy-handed* method. It flashes the computer screen when the user uses it for too long, before leading to a full screen flash.

Check out the :doc:`usage` section for further information.
